#  Bank Credit and Debit Dashboard

This project presents an interactive **Bank Credit and Debit Dashboard** created using **Microsoft Excel**.  
It provides clear insights into bank-wise and branch-wise transactions, monthly trends, and overall credit–debit performance.

---

##  Dashboard Snapshot

![Dashboard](https://github.com/madugondubhargavi/Bank-Analytics/blob/main/credit-debit-dashboard.png)

---

##  Features
- **Total Credit & Debit Summary**
- **Credit-to-Debit Ratio Calculation**
- **Monthly Transaction Trends (Line Chart)**
- **Branch-wise Total Transaction Amounts (Bar Chart)**
- **Bank-wise Credit and Debit Comparison (Clustered Column Chart)**
- **Credit vs Debit Trends Over Time**
- **Interactive Slicers for:**
  - Branch Name  
  - Bank Name  
  - Transaction Type (Credit / Debit)  
  - Year & Quarter  

---

##  Tools Used
- **Microsoft Excel**
- Pivot Tables  
- Slicers  
- Pivot Charts  
- Data Cleaning & Formatting  

---

##  Project Objective
To analyze and visualize credit and debit transactions across multiple banks and branches, making it easier to identify trends, compare performance, and derive business insights.
